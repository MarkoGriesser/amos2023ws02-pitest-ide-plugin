Use the pitest plugin, such as info.solidsoft.pitest or pl.droidsonroids.pitest, in the build file.

If gradle is used as a build system, utilize our companion Gradle plugin pitmutationmate-override-plugin in the build file.


Add the run configuration to initiate the pitest run:

Run → Edit Configurations → Add new configuration → PITmutationmate

The default options in the run configuration should be used.


To start a pitest run, you have several options:

  1. Click on the play button next to the class.
  2. Right-click inside the class in the editor.
  3. Right-click on the class name in the Project tool window.
